# Core JavaScript Demos

A small demo repository with three interactive JavaScript examples that cover:

- variables and data types
- functions and scope
- DOM manipulation and event handling
- arrays and objects
- async callbacks and promises

## How to use

1. Open `index.html` in your web browser.
2. Try the Counter demo, To-Do demo, and Quiz demo.

## Demo overview

- **Counter Demo**: increment and decrement a counter with DOM updates.
- **To-Do List Demo**: add tasks, mark them done, and remove them using an array of objects.
- **Quiz & Async Demo**: load a quiz question asynchronously using callbacks and promises.
