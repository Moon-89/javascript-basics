# JS Mini Apps Hub

A small collection of **three vanilla-JavaScript mini apps**, each on its own page, covering the core of front-end JS:

- variables and data types
- functions and scope
- DOM manipulation and event handling
- arrays and objects
- async callbacks and promises
- `localStorage` persistence

No frameworks, no build step — **plain JavaScript only**.

## How to use

1. Open `index.html` in your web browser.
2. Use the top navigation, or the cards on the home page, to open any app.

## Apps

| App | Files | Features |
|-----|-------|----------|
| **Home hub** | `index.html` | Landing page with cards linking to each app |
| **Counter** | `counter.html` + `counter.js` | Adjustable step size, +/− controls, reset, keyboard (↑/↓), live action-history log |
| **To-Do List** | `todo.html` + `todo.js` | Add/complete/remove, All/Active/Completed filters, live counts, clear-completed, `localStorage` persistence |
| **Quiz** | `quiz.html` + `quiz.js` | Four questions, progress bar, live score, submit & instant restart, answer highlighting |

`styles.css` is a shared stylesheet used by every page.
